# AI-Generated Curriculum Mobile App

An intelligent mobile application that automatically generates **structured learning curricula for highly specialised topics** by searching, analysing, and organising knowledge from across the web.

This project was developed as part of the **AI-Generated Curriculum assignment**, which explores how AI agents can transform fragmented internet knowledge into a guided learning experience.


## Features

- AI-generated learning paths
- Automated web search for relevant learning resources
- Content analysis and summarisation
- Structured curriculum generation
- Mobile learning interface
- Step-by-step guided learning experience

## Project Architecture

The project is divided into two main components:

### Frontend (iOS Mobile App)

The frontend is built using **Swift** and provides the user interface for interacting with the system.

Responsibilities:

- Topic search input
- Display generated curriculum
- Navigate lessons
- Track learning progress
- Communicate with backend APIs

Technologies:

- Swift
- iOS SDK
- REST API integration


### Backend (AI Service)

The backend handles **AI processing and curriculum generation**.

Responsibilities:

- Web search for relevant learning resources
- Content extraction
- AI analysis and summarisation
- Curriculum structure generation
